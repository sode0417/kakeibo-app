# 家計簿アプリ: 改善計画

## 概要
このディレクトリでは、家計簿アプリの改善計画に関する個別の Issue を管理しています。各 Issue は番号付きの Markdown ファイルとして保存され、詳細な改善内容と対応方針が記載されています。

---

## 親イシュー: 全体の改善計画

- **概要**:

  - 再利用性の向上。

### 2. バックエンドの改善
